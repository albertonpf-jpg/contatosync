Novo Setup

Guia completo passo a passo (30 min)

🚀 O que você vai fazer:

Configurar o ContatoSync na estrutura DO CLIENTE (GitHub dele + Railway dele) para funcionar com o WhatsApp dele.

O CLIENTE vai precisar ter:

  • ✅ Conta GitHub (você vai criar no nome dele - grátis)
  • ✅ Conta Railway (você vai criar no nome dele - grátis)
  • ✅ Conta Google Cloud (você vai criar no email dele - grátis)
  • ✅ Número do WhatsApp (do cliente)
  • ⏱️ 30 minutos do seu tempo para fazer o setup

⚠️ IMPORTANTE: Tudo será criado nas contas DO CLIENTE, não nas suas!

1

Criar Conta GitHub do Cliente

Criar conta e repositório no nome DO CLIENTE

Criar conta GitHub do cliente:

  1. Acesse github.com/signup
  2. Use o email do cliente para criar a conta
  3. Username: nome-do-cliente ou empresa-cliente
  4. Senha: Anote a senha que o cliente escolher
  5. Verifique o email do cliente e confirme a conta

Criar repositório ContatoSync:

  1. Abra github.com/albertonpf-jpg/contatosync
  2. Clique no botão verde "Use this template"
  3. Selecione "Create a new repository"
  4. ⚠️ IMPORTANTE: Faça login com a conta DO CLIENTE que você acabou de criar
  5. Nome do repositório: contatosync (pode usar nome simples)
  6. Marque como Private (privado)
  7. Clique em "Create repository"
✅ Resultado: Repositório criado em github.com/[usuario-do-cliente]/contatosync
⚠️ Note que está na conta DO CLIENTE, não na sua!
2

Criar Conta Railway do Cliente

Criar conta e fazer deploy no nome DO CLIENTE

Criar conta Railway do cliente:

  1. Acesse railway.app
  2. Clique em "Login" → "Login with GitHub"
  3. ⚠️ IMPORTANTE: Use a conta GitHub DO CLIENTE que você criou no passo 1
  4. Autorize o Railway a acessar o GitHub

Fazer deploy do ContatoSync:

  1. Clique em "New Project"
  2. Selecione "Deploy from GitHub repo"
  3. Procure e selecione: contatosync (do cliente)
  4. Aguarde 2-3 minutos (Railway vai instalar automaticamente)
  5. Quando aparecer "Success", clique no projeto
  6. Vá em Settings → Generate Domain
  7. Copie a URL gerada (ex: contatosync-production.up.railway.app)
✅ Resultado: Sistema online em https://xxxxx.up.railway.app
⚠️ Guarde essa URL! Está na conta Railway DO CLIENTE!
3

Criar Projeto Google Cloud do Cliente

Criar projeto no email DO CLIENTE

  1. ⚠️ IMPORTANTE: Faça login no Google Console com o email DO CLIENTE
  2. Acesse console.cloud.google.com
  3. Clique em "Selecionar projeto" → "Novo projeto"
  4. Nome do projeto: ContatoSync
  5. Clique em "Criar"
  6. Aguarde 10-20 segundos até aparecer notificação
  7. Clique em "Selecionar projeto" e escolha o ContatoSync
💡 Importante: O projeto Google Cloud DEVE estar no email do cliente para funcionar corretamente. Se você criar no seu email, os contatos vão para a SUA conta Google, não a do cliente!
4

Ativar People API

Permitir que o sistema salve contatos no Google

  1. No Google Cloud Console, procure por "People API" na barra de busca
  2. Clique em "People API" nos resultados
  3. Clique no botão azul "Ativar"
  4. Aguarde 5-10 segundos (vai ativar automaticamente)
✅ Resultado: People API ativada
5

Criar Credenciais OAuth 2.0

Gerar chaves de acesso ao Google

  1. No menu esquerdo, clique em "Credenciais"
  2. Clique em "Criar credenciais" → "ID do cliente OAuth"
  3. Se pedir para configurar tela de consentimento:
    • Selecione "Externo"
    • Nome do app: ContatoSync
    • Email de suporte: email do cliente
    • Clique em "Salvar e continuar" 3 vezes
    • Volte para "Credenciais"
  4. Tipo de aplicativo: "Aplicativo da Web"
  5. Nome: ContatoSync Web
  6. URIs de redirecionamento autorizados:
    Adicione a URL do Railway + /auth/google/callback
    Exemplo: https://xxxxx.up.railway.app/auth/google/callback
  7. Clique em "Criar"
  8. Copie o ID do cliente e Chave secreta do cliente
✅ Guarde essas 2 informações:
• Client ID (parece com: 123456-abc.apps.googleusercontent.com)
• Client Secret (parece com: GOCSPX-abc123)
6

Configurar Variáveis no Railway

Conectar tudo

  1. Volte para o Railway (railway.app)
  2. Abra o projeto do cliente
  3. Clique em "Variables"
  4. Clique em "Raw Editor"
  5. Cole este template e preencha os dados: 
    GOOGLE_CLIENT_ID=cole_aqui_o_client_id_do_passo_5
GOOGLE_CLIENT_SECRET=cole_aqui_o_client_secret_do_passo_5
GOOGLE_REDIRECT_URI=https://sua-url-railway.up.railway.app/auth/google/callback
PORT=8080
SESSION_SECRET=gere_uma_senha_aleatoria_de_32_caracteres
  6. Clique em "Update Variables"
  7. Aguarde 1-2 minutos (Railway vai reiniciar automaticamente)
💡 Para gerar SESSION_SECRET: Use um gerador de senhas ou digite 32 caracteres aleatórios
7

Conectar WhatsApp do Cliente

Fazer a sincronização funcionar

  1. Abra a URL do Railway no navegador
  2. Você verá uma página com QR Code do WhatsApp
  3. Peça para o cliente abrir o WhatsApp dele
  4. WhatsApp → Menu (3 pontos) → "Aparelhos conectados"
  5. Clique em "Conectar um aparelho"
  6. Escaneie o QR Code da tela
  7. Aguarde 5-10 segundos
  8. Deve aparecer "WhatsApp Conectado ✅"
✅ Resultado: WhatsApp conectado e sincronizando
8

Conectar Google Contacts

Autorizar salvamento de contatos

  1. Na mesma página, clique em "Conectar Google"
  2. Vai abrir tela de login do Google
  3. Faça login com a conta Google do cliente
  4. Vai aparecer aviso "Google não verificou este app"
  5. Clique em "Avançado" → "Ir para ContatoSync (não seguro)"
  6. Marque as permissões e clique em "Continuar"
  7. Deve redirecionar e aparecer "Google Conectado ✅"
⚠️ Importante: O aviso de "não verificado" é normal. É porque o app não passou pela verificação do Google (que custa US$ 15-220 e demora 4-6 semanas). O app é seguro!
✓

Setup Completo!

Agora vem a parte final

Teste se está funcionando:

  1. Peça para alguém enviar uma mensagem no WhatsApp do cliente
  2. Aguarde 10-30 segundos
  3. Abra o Google Contacts do cliente (contacts.google.com)
  4. Verifique se o contato foi salvo automaticamente

Adicione o cliente no painel:

  1. Volte para a aba "Clientes" deste painel
  2. Clique em "Novo Cliente"
  3. Preencha todos os dados:
    • Nome, email, WhatsApp
    • Plano contratado
    • URL do Railway
    • Data de instalação (hoje)
    • Valor do setup cobrado
  4. Clique em "Salvar Cliente"
🎉 Parabéns! Cliente configurado e funcionando!
